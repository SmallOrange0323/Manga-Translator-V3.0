# Manga Translator V3.0 隱私權政策 (Privacy Policy)

**最後更新日期：2026-04-27**

本隱私權政策旨在說明 **Manga Translator V3.0**（以下簡稱「本軟體」）如何處理您的資料。

## 1. 資訊蒐集與使用
本軟體致力於保護您的隱私。本軟體本身**不會**蒐集、儲存或向第三方分享任何您的個人識別資訊（如姓名、地址、電子郵件、電話號碼或身分證字號）。

## 2. 翻譯資料處理與傳輸
為了提供翻譯服務，本軟體會執行以下資料處理流程：
*   **圖片資料處理**：當您主動點擊翻譯按鈕時，本軟體會將您選取的網頁圖片內容轉換為資料流。
*   **第三方 API 傳輸**：上述圖片資料會經由加密連線傳送至 **Google Gemini API** 進行文字辨識（OCR）與翻譯。
*   **API 金鑰安全**：您在設定頁面輸入的 API 金鑰僅儲存於您個人裝置的 `chrome.storage.local` 安全空間中。本軟體僅在與 Google 官方 API 通訊時使用該金鑰，絕不會將金鑰傳送至任何其他伺服器。

## 3. 第三方服務聲明
本軟體使用 Google Gemini API 作為核心翻譯引擎。當資料傳送至 Google 伺服器後，其資料處理與隱私保護方式受 [Google 隱私權政策](https://policies.google.com/privacy) 之約束。

## 4. 本地資料儲存
本軟體所有的設定值（如主題、模型選擇、API Key）與翻譯結果快取皆儲存在使用者的本地設備中。本軟體開發者無法存取、檢視或修改這些儲存於您本地裝置的資料。

## 5. 政策變更
若本隱私權政策有任何變動，我們將更新此文件。建議您定期檢視。

## 6. 聯絡我們
如果您對本隱私權政策或本軟體有任何問題，請透過擴充功能商店頁面提供的聯絡資訊與開發者聯繫。
